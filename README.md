# fighting-game

A fighting game I am working on utilizing HTML canvas and vanilla JavaScript. This is a two player game where each player controls a character and duke it out old school style!

Controls for P1:
Jump - W
Movement - A/S
Attack - E
Block - Q
Super - A + S
Slide - S + D
Jump Attack - W + E

Controls for P2:
Jump - I
Movement - J/L
Attack - U
Slide - J + K
